GitLab - Runner for DevOps. when code checked in, runner gets triggered.

Configuration:
> git config --global user.name ""
> git config --global user.email @.com
> git config --list

>mkdir FirstRepo
>cd FirstRepo
>git init

Creates a .git file.

>create a file.
>git status
lists the changes.

>git add .
moves file to staged area.

to unstage or remove
>git remove

>git commit -m "First repo commit"

>git log
log of changes made.

>git remote add origin https://github.com/bddprasad/FirstRepo.git
>git remote -v

>git push -u origin master

>git clone server_repo_url

>git reset --soft 
removes the commit comments but retain the content.
